## 技术实现细节

### 后端实现

1. **项目结构**

   ```plaintext
   com.wuyi.mall/
   ├── entity/        # 实体类
   ├── dto/           # 数据传输对象
   ├── controller/    # 控制器，处理HTTP请求
   ├── service/       # 业务逻辑层
   ├── repository/    # 数据访问层
   ├── config/        # 配置类
   └── security/      # 安全相关
   ```

2. **技术点**

   - 使用 Spring Boot Starter Web 构建 RESTful API
   - Spring Security + JWT 实现身份认证与授权
   - Spring Data JPA 操作数据库

###  前端实现

1. **项目结构**

   ```
   src/              # 主代码目录
   ├── assets/       # 资源文件
   ├── components/   # Vue 组件
   ├── router/       # 路由配置
   ├── utils/        # 工具类
   ├── views/        # 页面组件
   │   ├── User/     # 用户相关页面
   │   └── admin/    # 管理员相关页面
   ├── App.vue       # 根组件
   └── main.js       # 入口文件
   ```

2. **技术点**

   - Vue 3 Composition API 组织组件逻辑
   - Vue Router 实现路由管理
   - Axios 与后端 API 通信
   - 响应式布局设计，适配不同设备

### 数据库设计

- 主要表结构：用户表、商品表、分类表、订单表、订单项表、地址表等

## 遇到的问题与解决方案

### 跨域问题

- **问题**：Vue与Spring Boot域名/端口不同导致跨域请求被阻止
- **解决方案**：在后端配置跨域资源共享（CORS），允许前端域名的请求

### 表单验证

- **问题**：前端表单提交的数据有效性校验不充分
- **解决方案**：实现前后端双重验证，前端通过Vue指令验证，后端通过注解验证
